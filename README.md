# Local Markets Platform – Microservices Architecture ☁️

## About
Backend platform designed using a **microservices architecture** to manage markets, sellers, users, favorites, ratings, and notes.
This project focused on service separation, scalability, and communication between independent services.

## Microservices
- Users & Authentication
- Fairs Management
- Favorites
- Ratings
- Notes
- Notifications

## Technologies
- Node.js
- MongoDB
- GraphQL(markets service)
- Python (Notes service)
- Java (Favorites service)
